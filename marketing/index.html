Troubleshooting

Location not recording

  1. Settings → Privacy → Location Services → TimeTrails → Allow “Always” and “Precise Location”.
  2. Enable Background App Refresh for TimeTrails.
  3. Low Power Mode may pause tracking; try disabling it.
  4. Reboot your iPhone, then open TimeTrails once.

Battery usage feels high

  1. Use “Smart sampling” in Settings → Tracking.
  2. Exclude very short stays in Settings → Filters.
  3. Update to the latest app version.

Export & Import

Export
Settings → Data → Export → choose GPX/CSV and date range. Files save to the Files app or can be shared.
Import
From Files or the Share Sheet: choose a GPX file, then open with TimeTrails.

FAQs

        

            Does TimeTrails track me all the time?
            
You control what’s collected. Recording uses iOS location services and can be paused anytime. Data stays on-device unless you enable sync.

        

        

            What permissions are required?
            
Location (Always & Precise) for background trails, Motion for better activity detection, Photos if you import media, and Notifications for reminders—all optional except Location for recording.

        

        

            How do I delete my data?
            
In Settings → Data → Delete local data. If iCloud sync is enabled, disable it first, then delete local data across each device as needed.
